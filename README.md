-게임 소개 영상 : https://youtu.be/nBqD8GQ6k7A

주요 특징:
- **물체 밀기**: 플레이어는 자기의 힘을 사용해서 특정 오브젝트를 밀거나 당길 수 있습니다.

- **물체**: 오브젝트들은 자신만의 힘을 가지고 있습니다.
